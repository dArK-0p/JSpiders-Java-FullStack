# Operators

- Special *symbol* which is used to perform a **specific** operation.
- Types:
  - Arithmetic(`+` `-` `*` `/`).
  - Relational(`>` `<` `>=` `<=`).
  - Concatenation(`||`).
  - Logical(`AND` `OR` `NOT`).
  - Comparison(`==` `!= or <>`).
  - Special Operators(`IN` `NOT IN` `IS` `IS NOT` `LIKE` `NOT LIKE` `BETWEEN` `NOT BETWEEN`).
  - Sub-Query Operators(`ALL` `ANY` `EXIST` `NOT EXIST`).
