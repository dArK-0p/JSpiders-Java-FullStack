# JSpiders Java Full Stack Repository

This private repository is a structured workspace for the **Java Full Stack Developer** course. It contains carefully organized code samples, conceptual notes, and diagrams relevant to topics covered in Core Java, SQL, and Web Technologies.

---

## 📁 Repository Structure

```
JSpiders-Java-FullStack/
├── Core Java/        # Java programs, notes, and supporting images
│   ├── Code/         # Source and compiled Java files
│   └── Notes/        # Markdown notes with images for reference
├── SQL/              # SQL notes and theory
│   └── Notes/        # SQL topic-wise markdown notes
└── WebTech/          # Notes and placeholder for web development content
    ├── Code/
    └── Notes/
```

---

## ✅ Highlights

* **Clean Codebase**: Java classes follow camelCase naming and are grouped by functionality.
* **Notes with Diagrams**: Comprehensive Markdown notes accompanied by well-labelled diagrams under `Images/`.
* **Separation of Concerns**: Code and notes are separated to maintain clarity.
* **Consistent Image Linking**: All images are referenced using the path format:

  `/home/darkop/Documents/JSpiders-Java-FullStack/Core%20Java/Notes/Images/<filename.ext>`
